# Think About It — Discussion Responses

This folder contains discussion responses for the "Think About It" prompts embedded in each lecture week's companion reading and slide deck. These prompts are designed to spark 2–3 minutes of class discussion during the live lecture.

**These responses are released after class** so students engage with the prompts first without seeing the "answer." There is no single right answer to any of these. The responses below highlight key tensions, multiple perspectives, and connections to course themes.
